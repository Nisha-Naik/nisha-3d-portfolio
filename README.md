# 🌟 About the Project

Welcome to my personal 3D portfolio.

This project is more than just a portfolio website—it's a digital experience designed to showcase my journey as a Computer Science Engineering student, Full Stack Developer, AI enthusiast, and hackathon participant.

The portfolio combines modern web technologies, immersive animations, elegant UI, and responsive design to create an engaging experience for recruiters, developers, and collaborators.

Every interaction, animation, and transition has been carefully crafted to reflect both creativity and technical excellence.

# ⚙️ Getting Started

## Clone the Repository

```bash
git clone https://github.com/Nisha-Naik/nisha-3d-portfolio.git
```

## Navigate to the Project

```bash
cd nisha-3d-portfolio
```

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
npm run dev
```
